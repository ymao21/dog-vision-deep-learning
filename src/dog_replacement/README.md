# Dog Replacement Source Package

Recommended future modules:

- `detect.py`: YOLO dog detection and pose estimation.
- `segment.py`: YOLO/SAM mask generation and refinement.
- `pairing.py`: pose index, candidate search, feasibility scoring.
- `warp.py`: similarity and piecewise affine alignment.
- `draggan.py`: face inversion, PTI, and latent editing adapters.
- `diffusion.py`: local inpainting and seam repair.
- `harmonize.py`: post-generation color and boundary refinements.
- `evaluate.py`: cycle-consistency metrics and ablation reporting.

The current implementation lives in `notebooks/dog_replacement/`.
