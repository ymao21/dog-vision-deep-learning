# Classification Source Package

Recommended future modules:

- `dataset.py`: Stanford Dogs indexing, labels, transforms, splits.
- `model.py`: ResNet-50 multi-task architecture.
- `train.py`: training loop, validation, checkpointing.
- `evaluate.py`: test metrics and per-class analysis.
- `infer.py`: single-image and batch inference helpers.

The current implementation lives in `notebooks/classification/01_multitask_classification.ipynb`.
