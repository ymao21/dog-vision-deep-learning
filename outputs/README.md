# Outputs

Generated outputs are excluded from Git by default.

For a public portfolio repository, commit only a small curated set of examples:

```text
outputs/examples/
├── classification_demo.png
├── dog_replacement_before_after.png
├── inpainting_comparison.png
└── cycle_consistency_metrics.csv
```

Recommended examples should be permission-safe, visually clear, and small enough for GitHub to render quickly.
