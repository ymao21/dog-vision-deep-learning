## Portfolio Blurb

Classifying and Reimagining Dogs with Deep Learning is an applied computer vision project that combines discriminative and generative modeling. It includes a ResNet-50 multi-task classifier, a pose-aware dog replacement pipeline using YOLO and SAM-style segmentation, and CNN/diffusion inpainting workflows for restoring missing image regions. The project emphasizes end-to-end ML pipeline design, model evaluation, and practical deployment considerations.
